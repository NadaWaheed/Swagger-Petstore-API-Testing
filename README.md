# Swagger Petstore API Testing (Postman Collection)

Simple Postman collection for practicing **API testing** using the Swagger Petstore API.

The collection covers basic CRUD operations for:
- **Pet** (add, find, update, delete, upload image)
- **Store** (place order, get order, inventory, delete order)
- **User** (create user, login, logout, update, delete)  

It includes both **valid** and **invalid** requests to practice:
- Status code checks (200, 400, 404, etc.)
- Simple response validation
- Dynamic data using Postman environment variables (e.g. random `tag_id` and `tag_name` for pets).
